# 🎬 youtube-slicer

**youtube-slicer** 是一個 Google Chrome 擴充功能，  
讓語言學習者可以在 YouTube 影片中**標記時間片段**，並快速重複播放特定區段，  
非常適合用於：

- 語言聽力練習  
- 跟讀 / Shadowing  
- 句型、發音、單字反覆練習  

---

## ✨ 功能特色

- ⏱️ **即時紀錄播放時間點**
- 🔁 **快速重複播放指定片段**
- 📌 **以分段標記方式管理影片內容**
- 🎧 專為語言學習情境設計，操作簡單直覺

---

## 🧩 使用方式

### 1️⃣ 安裝擴充功能（開發者模式）

1. 開啟 Google Chrome  
2. 前往 `chrome://extensions/`
3. 右上角開啟 **「開發者模式」**
4. 點擊 **「載入未封裝項目」**
5. 選擇本專案的資料夾

---

### 2️⃣ 在 YouTube 上使用

1. 開啟任一 YouTube 影片
2. 點擊 Chrome 工具列中的 **youtube-slicer** 擴充功能圖示
3. 選擇 **「啟動紀錄時間」**
4. 在影片播放過程中標記你想反覆練習的時間點
5. 透過標記快速重播指定片段

---

## 🎯 適合對象

- 語言學習者（英語 / 日語 / 韓語 / 其他外語）
- 需要反覆聆聽特定句子的學習者
- 使用 YouTube 作為主要學習素材的人
